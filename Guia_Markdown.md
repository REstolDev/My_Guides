## Textos

### Negrita

'__ __' o '** ** '

__negrita__

**negrita**

Esto **en negrita**, esto no

### Cursiva

'_ _' o '* *'

_cursiva_

*cursiva*

### Combinado

'*** ***'

Esto es ***negrita y cursiva***


## Listas
### UL
' * o - o +'
* Lista
- Lista2
+ Lista3
    + Lista 4

### OL
'Numero y punto ex. '1.''
1. Lista 1
2. Lista 2

### Combinacion Listas
1. Lista 1
    - Lista 1.1
    - Lista 1.2

    </tr>
</table>
